# Desktop App — Toward Perfection DM Manager

**Статус:** Планирование  
**Начало разработки:** 2026-05-05  
**Цель:** Заменить Telegram-интерфейс бота на нативное Mac-приложение

---

## Зачем

Telegram неудобен как рабочее место: всё в одном чате, нет структуры, некомфортно проводить там рабочие часы. Нужно отдельное приложение, которое открываешь → работаешь → закрываешь.

---

## Что строим

**Electron-приложение для Mac** с тремя задачами:
1. Показывать все Instagram DM в удобном интерфейсе
2. Генерировать ответы через Claude (обученный на стиле тренера)
3. Постепенно накапливать стиль и двигаться к автономности

---

## Стек

| Слой | Технология | Почему |
|------|-----------|--------|
| Оболочка | Electron | Mac-приложение, официальный API, без ban-риска |
| UI | React + Tailwind | Быстро, гибко, легко менять |
| Шрифт | Geist | Уже используется на сайте, единый стиль |
| Дизайн | Стиль сайта | Те же токены что на сайте, тёмная тема по умолчанию, переключатель день/ночь |
| AI основной | Claude (Anthropic SDK) | Уже подключён, генерирует варианты ответов |
| AI контекст | OpenAI API | История переписок из ChatGPT → обогащение стиля |
| Бэкенд | Node.js + Express | Уже написан, переносится без изменений |
| Instagram | Meta Graph API | Официальный, уже работает |
| Хранилище | JSON-файлы локально | Просто, надёжно, без лишних зависимостей |

## Фазы разработки

### Фаза 1 — Скелет приложения [ ]
- [ ] Инициализировать Electron + React проект (`electron-forge` или `electron-vite`)
- [ ] Настроить Tailwind с дизайн-системой (CSS-переменные)
- [ ] Реализовать переключатель тёмная/светлая тема (кнопка в правом верхнем углу)
- [ ] Три-колоночный layout без данных (статичный)
- [ ] Кастомный titlebar (убрать стандартный macOS, сделать свой)

### Фаза 2 — Подключение бэкенда [ ]
- [ ] Перенести `index.js` логику в Electron main process
- [ ] Instagram webhook → принимать входящие DM
- [ ] `conversations.json` → загружать и отображать в Inbox
- [ ] Список диалогов с цветовыми статусами (Новое / Отвечено / Клиент / Игнорировать)
- [ ] Открытие диалога → показ истории переписки

### Фаза 3 — AI-генерация ответов [ ]
- [ ] Перенести `generateSuggestions` в Electron
- [ ] Показывать 3 варианта в правой панели
- [ ] Показывать % совпадения со стилем
- [ ] Кнопки «Выбрать 1/2/3» → текст переходит в поле ввода
- [ ] «Написать своё» → чистое поле ввода
- [ ] Confirm → отправка в Instagram → статус «Отвечено»
- [ ] Каждый одобренный ответ → сохраняется в `style_profile.json`

### Фаза 4 — Импорт стиля [ ]
- [ ] **ChatGPT Export:** парсер `conversations.json` из экспорта OpenAI
  - Извлекать все твои сообщения клиентам
  - Фильтровать нерелевантные (технические, не про фитнес)
  - Добавлять в `style_profile.json` как примеры стиля
- [ ] **Meta Archive:** парсер архива Instagram DM
  - Скачать из Meta: Settings → Your information → Download your information
  - Извлечь исходящие сообщения из JSON-архива
  - Добавить в стиль-профиль
- [ ] UI для импорта: экран настроек с кнопками «Загрузить ChatGPT экспорт» / «Загрузить Meta архив»
- [ ] Показывать счётчик примеров стиля (было 0 → стало 847)

### Фаза 5 — Полировка UX [ ]
- [ ] Горячие клавиши: `⌘1/2/3` → выбрать вариант, `⌘Enter` → отправить
- [ ] Уведомления macOS при новом DM
- [ ] Поиск по диалогам
- [ ] Фильтры: только Новые / только Клиенты
- [ ] Анимации переходов (плавно, как в Linear)
- [ ] App icon для macOS Dock

### Фаза 6 — Автономность (будущее) [ ]
- [ ] Режим «Авто-черновик»: Claude составляет ответ, ты видишь его утром
- [ ] Режим «Авто-отправка»: Claude отвечает сам, ты видишь лог
- [ ] Переключатель режима в настройках (всегда `false` по умолчанию)
- [ ] Дневной лимит авто-ответов (защита от ошибок)
