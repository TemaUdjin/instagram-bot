# Сессия 2026-05-07 — Приложение работает полностью

## Статус: РАБОЧИЙ ПРОДУКТ

Приложение запущено, отправляет в Instagram, Claude даёт экспертные ответы.

## Как запустить (каждый раз)

```bash
# Терминал 1 — сервер
cd ~/Projects/toward-perfection/instagram-bot/dm-app
npm run server

# Терминал 2 — приложение
cd ~/Projects/toward-perfection/instagram-bot/dm-app
source ~/.cargo/env && cargo tauri dev
```

## Что работает сейчас

- Реальные Instagram DM в inbox (24 активных диалога)
- Фильтр 24ч — показывает только тех кому можно ответить
- Фильтры: Все / Новые / Отвечено / Клиенты
- Кнопка "скрыть" — убирает ненужные диалоги (запоминается)
- Отправка в Instagram — реально доходит
- Авто-обновление диалога каждые 8 секунд
- Кнопка "✦ Спросить Claude" — генерирует варианты ответа
- Живой чат с Claude — принимает коррекции ("напиши короче", "на английском")
- Экспертная база знаний: хэндстенд, йога, мобильность (5700+ символов)
- Скилл-промпт: человечный стиль, не бот
