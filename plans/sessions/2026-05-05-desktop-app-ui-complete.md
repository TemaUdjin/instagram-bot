# Сессия 2026-05-05 (вечер) — Desktop App UI завершён

## Задача
Построить полный UI десктоп-приложения на Tauri v2 + React.

## Что сделано

### Запущено приложение
- Tauri v2.11.0 + React 19 + Tailwind v4 + Geist шрифт
- Папка: `instagram-bot/dm-app/`
- Запуск: `cd dm-app && source ~/.cargo/env && cargo tauri dev`

### UI компоненты (все в `dm-app/src/components/`)

| Компонент | Что делает |
|-----------|-----------|
| `ActivityBar.tsx` | Левая панель иконок как в VS Code: Диалоги / Клиенты / Статистика / Стиль / Настройки |
| `TabBar.tsx` | Вкладки диалогов как в VS Code, с крестиком по hover |
| `Inbox.tsx` | Список диалогов: поиск (⌘K), счётчик ожидания (красный >2ч), follow-up статус |
| `Dialog.tsx` | Переписка: заметки о клиенте (✏️), follow-up reminder (🔔), ссылка на Instagram ↗ |
| `SuggestionsPanel.tsx` | Живой чат с Claude: варианты кликабельные, поле ввода, вставка скриншота (⌘V), выбор модели, прогресс-бар контекста |
| `ResizeHandle.tsx` | Перетаскиваемые разделители между панелями (подсвечиваются золотом) |
| `Titlebar.tsx` | Кастомный titlebar с drag-регионом и переключателем тема |

### Дизайн
- **Тёмная тема** (default): `#1e1e1e` фон, `#252526` панели, `#c8a96e` золото — VS Code стиль
- **Светлая тема**: `#f3f3f3` фон, `#e8e8e8` панели, `#a07840` золото
- Шрифт: Geist Variable
- Иконка приложения: чёрный круг + белый треугольник вверх

### Горячие клавиши
- `⌘K` — фокус на поиск
- `⌘[` / `⌘]` — переключение вкладок
- `⌘W` — закрыть вкладку
- `⌘V` в поле Claude — вставить скриншот из буфера

### Панель Claude (правая)
- Живой чат: варианты → кликаешь → выделяется → «Подтвердить и отправить»
- Пишешь "сделай мягче" → Claude переделывает (пока mock, Фаза 3 — реальный API)
- Выбор модели: Sonnet 4.6 / Opus 4.7 / Haiku 4.5
- Прогресс-бар контекста с токенами
- Кнопка 📎 + вставка скриншота `⌘V`
